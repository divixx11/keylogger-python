# Keyboard Monitor (Educational Project)

Description
This project is a simple keyboard event monitoring application built using Python and the `pynput` library.
It captures typed words while the program is running and saves each completed word on a new line in a log file.

1) This project is for educational purposes only.
2) It runs visibly in the foreground.
3) It stops when the ESC key is pressed.

Features
- Captures keyboard input while active
- Saves each word on a new line
- Handles backspace correctly
- Stops safely on ESC key
- No hidden or background behavior

Technologies Used
- Python 3
- pynput library
